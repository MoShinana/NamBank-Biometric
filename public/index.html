SECURITY ARCHITECTURE NOTE:
Why this is secure:
The public key stored in this database is mathematically useless without the matching private key. The private key is sealed inside the device's TPM (Trusted Platform Module) or Secure Enclave. It can never be exported. Even if this entire database is stolen, no attacker can authenticate as any user or reconstruct any fingerprint. This is the fundamental security advantage of WebAuthn/FIDO2 over traditional biometric template storage systems.
